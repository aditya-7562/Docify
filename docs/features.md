# Docify Features

Docify delivers a full Google Docsâ€“style experience with real-time collaboration,
rich editing, document management, and secure sharing. The sections below capture

## Real-Time Collaboration

- **Live Editing**: Multiple users can edit documents simultaneously with instant synchronization.
- **Live Cursors**: See where collaborators are working in real time.
- **Presence Indicators**: View active users and their avatars.
- **Offline Support**: Keep working offline with automatic sync when reconnected.
- **Conflict Resolution**: Automatic reconciliation powered by CRDTs.

## Rich Text Editor

- **Comprehensive Formatting**: Bold, italic, underline, strikethrough, and more.
- **Typography Controls**: Heading levels, font families, font sizes, and line heights.
- **Text Alignment**: Left, center, right, and justify.
- **Lists**: Bulleted, numbered, and task lists with checkboxes.
- **Tables**: Full table support with cells, headers, and rows.
- **Images**: Insert and resize images inline.
- **Links**: Automatic link detection and manual insertion.
- **Highlighting & Colors**: Multi-color highlights plus custom text/background colors.
- **Remove Formatting**: Quickly strip formatting from selections.

## Comments & Threading

- **Threaded Comments** anchored to specific text selections.
- **Floating Comment Sidebar** for browsing discussions.
- **Comment Resolution** workflows.
- **Real-Time Updates** so every participant sees comments instantly.
- **Notifications** to stay informed about document conversations.

## Document Management

- **Create Documents** from scratch or via professional templates (blank, software proposal, project proposal, business letter, resume, cover letter, general letter).
- **Organization Tools**: Folders, tags, starring, and pagination for fast navigation.
- **Search**: Full-text search across the workspace.
- **Document Renaming & Deletion** with safe flows.

## Sharing & Permissions

- **Role-Based Access Control**: Viewer, Commenter, and Editor roles.
- **Share Links** with customizable permissions and optional expiration.
- **Organization Support** for sharing inside teams.
- **User-Level Permissions** to grant, modify, or revoke access quickly.

## Version History

- **Snapshots**: Create manual snapshots with metadata (creator, time, description).
- **Restoration**: Revert to any previous state.
- **Chronological View** to explore historical changes.

## User Experience Highlights

- **Modern UI** using Tailwind CSS and shadcn/ui.
- **Keyboard Shortcuts** for power users.
- **Document Ruler** and adjustable margins.
- **Print Support** optimized for export.
- **Responsive Design** with dark mode, thoughtful loading states, and graceful error handling.

## Authentication & Security

- **Clerk Integration** for auth and organizations.
- **JWT Authentication** shared across Convex and Liveblocks.
- **Protected Routes** and rich user identity data.

## Development Tooling

- **Vitest** and **Testing Library** for automated tests.
- **ESLint** and **TypeScript** for consistent, type-safe development.
